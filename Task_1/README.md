Task 1 – Rule-Based Chatbot

This project implements a simple rule-based chatbot using Python. The chatbot responds to user input based on predefined rules and conditional statements.

Features
	•	Interactive command-line chatbot
	•	Keyword-based response system
	•	Continuous conversation loop
	•	Simple and structured logic

Technologies Used
	•	Python

How It Works
	1.	The program accepts user input from the terminal.
	2.	It checks for specific keywords using conditional statements.
	3.	Based on matching rules, it returns predefined responses.
	4.	The loop continues until the user exits the program.
