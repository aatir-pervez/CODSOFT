This project implements real-time face detection using OpenCV and a pre-trained Haar Cascade classifier.

## Features

- Real-time webcam face detection
- Image-based face detection
- Face counter display
- Automatic saving of detected faces
- Clean modular code structure

## Technologies Used

- Python
- OpenCV
- Haar Cascade Classifier

## How It Works

1. Convert frame to grayscale.
2. Use Haar Cascade model to detect faces.
3. Draw bounding boxes around detected faces.
4. Display total number of faces.
5. Save cropped face images.
